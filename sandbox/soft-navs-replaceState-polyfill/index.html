When you click the link, we will soft-navigate the document, and you should see a soft-nav and ICP performance entry in console.

        However, the history stack should not grow beyond a single entry, i.e., it will have "replaceState" semantics.

        To do this, the link will:
        - Custimize the `click` event and call preventDefault()
        - Similate 1s of network delay with a timer
        - Polyfill replaceState via:
          - register a `popstate` event with { once: true }
          - Call pushState
          - PreventDefault
